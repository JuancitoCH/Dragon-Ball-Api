# Dragon Ball Api

# Usage
- Create and complete the .env with the content of .env.example
- Run the Following command:
> Install all dependencies
> - npm i
> 
> Start The Api
> - npm run Start
> 
> Run in Dev Mode
> - npm run dev
> 
> Run All Test
> - npm run test

# Tools
- Typescript
- Nodejs
- Express
- Git
